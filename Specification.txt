Проект на express, отдающий одну страницу

-- Разметка страницы
Первый блок содержит в себе два вложенных блока, расположенных горизонтально друг за другом
  Вложенный блок 1 - input для ввода ID пользователя
  Вложенный блок 2 - иконка календаря, по нажатию на которую открывается календарь для выбора даты
Второй блок содержит в себе два вложенных блока, расположенных горизонтально друг за другом
  Вложенный блок 1 - сводка
  Вложенный блок 2 - графики

-- Сводка
Сводка это список ключей и значений. 
Пример того, как это выглядит: http://screen.ex2b.co/2017-12-12_135122.jpg
Одна строка - это ключ(CPA/ДОХОД) и значение (404,13Р/4,647,084Р)

-- График
Использовать можно любую библиотеку, на твой выбор
Один шаг по оси X - это один день месяца. В графике отображаем последние 30 дней
На одном графике имеется сразу две оси Y, одна отображает кол-во, другая сумму в рублях (пример того, что значит 2 оси Y: http://screen.ex2b.co/2017-12-12_142350.jpg)

-- Данные
Данные берутся из таблицы stats (дамп таблицы с данными приложен к заданию)
Колонка `partner_id` определяет id пользователя которому принадлежат записи (используем её для выбора записей пользователя из input'а в первом блоке на странице)
Колонка `event` определяет тип записи
Колонка `date` определяет день записи
Колонка `event_value` определяет значение записи (сумма в рублях, например)

-- Значения которые должны быть в сводке и таблице
"Переходы" - записи с `event` = 'LINK_VISITOR'
"Регистрации" - записи с `event` = 'REGISTRATION'
"Доход" - записи с `event` = 'PAYMENT'

Значения "Переходы" и "Регистрации" - это кол-во
Значение "Доход" - это сумма денег

-- Дополнения
- Если ID юзера не введен - график и сводку не выводим, пишем сообщение "Введите ID пользователя"
- При изменении фильтров(id пользователя\выбор даты на календаре) - новые данные запрашиваются без обновления страницы. То есть страница грузится только один раз, всё дальнейшая работа(загрузка данных, происходит без обновления
- В календаре должен выбираться период, а не один день (пример: http://screen.ex2b.co/2017-12-12_143318.jpg)
- Суммы денег во всех местах где они отображаются(включая label в графике) должны форматироваться (пример: http://screen.ex2b.co/2017-12-12_143401.jpg)
